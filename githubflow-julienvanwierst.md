Qu'est-ce qu'une Pull Request ?
Une Pull Request est une demande faite par un développeur pour intégrer ses modifications de code dans une branche principale
Qu'est-ce qu'une Code Review ?
La Code Review (revue de code) est le processus d'examen du code d'un collègue avant qu'il soit intégré. Elle se déroule généralement dans le cadre d'une PR (Pull Request).
